# Анализ рынка IT-вакансий в Power BI

## О проекте

В этом проекте я собрал аналитический дашборд по рынку IT-вакансий на основе объединённой выборки вакансий.

Данные для анализа были собраны с сайта hh.ru и содержат IT-вакансии за осень 2022 года.

Основная идея проекта — не просто показать набор графиков, а ответить на несколько понятных вопросов:
как распределяются вакансии по опыту,
как опыт влияет на нижнюю границу зарплаты,
какие типы занятости встречаются чаще,
насколько работодатели указывают зарплату,
и как выглядит структура выборки по работодателям и регионам.

Проект сделан в Power BI на основе нескольких исходных таблиц с вакансиями, которые были объединены в одну витрину данных и очищены перед анализом.

---

## Что я делал

### Подготовка данных

В качестве основы использовались три отдельные выборки IT-вакансий, собранные с сайта hh.ru за осень 2022 года:

`vacancies_part_356` — выборка на 356 строк
`vacancies_part_604` — выборка на 604 строки
`vacancies_part_858` — выборка на 858 строк

После объединения была собрана итоговая таблица для анализа.

С данными я сделал следующее:

Привёл названия и структуру столбцов к единому виду  
Объединил несколько выборок в одну таблицу  
Удалил дубликаты и обработал пропуски  
Преобразовал поле заработной платы в удобный для анализа формат  
Выделил нижнюю границу зарплаты в рублях в отдельный числовой столбец  
Проверил типы данных и исправил некорректные значения  
Подготовил итоговую таблицу для построения дашбордов

---

## Что показывает дашборд

В итоговой версии проекта я собрал несколько отчётов, каждый из которых отвечает на отдельный аналитический вопрос.

### 1. Обзор рынка вакансий

На этой странице показаны основные характеристики выборки:

общее количество вакансий  
количество компаний  
средняя минимальная зарплата  
медианная минимальная зарплата  
зависимость нижней границы зарплаты от опыта работы  
распределение вакансий по опыту  
средняя минимальная зарплата по типу занятости  

Эта страница помогает быстро понять общую структуру рынка и увидеть, как требования к опыту и формат занятости связаны с уровнем оплаты.

### 2. Работодатели, зарплаты и вакансии

На этой странице собраны графики, которые показывают:

топ компаний по числу вакансий  
распределение вакансий по диапазонам зарплаты  
долю вакансий с указанной и неуказанной зарплатой  
структуру вакансий по типу занятости  

Эта страница позволяет понять, насколько рынок прозрачен по зарплатам и какие работодатели и форматы занятости заметнее представлены в выборке.

### 3. География размещения вакансий

На отдельной странице вынесена карта, которая показывает распределение вакансий по регионам.

Это дополнительный слой анализа, который позволяет оценить, насколько широко распределены вакансии по территории и где сосредоточена основная активность работодателей.

---

## Аналитическая польза проекта

Этот проект полезен как пример работы с реальными табличными данными и построения BI-отчёта, который помогает быстро ответить на прикладные вопросы по рынку вакансий.

По итоговому дашборду можно делать выводы, например:

какие уровни опыта встречаются чаще всего  
как меняется нижняя граница зарплаты в зависимости от опыта  
какие типы занятости связаны с более высокой оплатой  
насколько часто работодатели указывают зарплату  
какие компании и регионы активнее размещают вакансии
